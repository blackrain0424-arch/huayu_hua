## 2. 数据流图 (Sequence Diagram)

```mermaid
sequenceDiagram
    actor U as 用户
    participant F as Flutter 前端
    participant S as Supabase
    participant EF as Edge Function
    participant AI as DeepSeek / Qwen-VL
    participant L as 本地离线引擎

    Note over U,L: === 在线模式：AI 文字对话 ===
    U->>F: 输入 "牡丹的花语是什么？"
    F->>S: 调用 Edge Function (chat)
    S->>EF: 转发请求
    EF->>AI: 调用 DeepSeek API
    AI-->>EF: 返回 AI 回复
    EF-->>S: 返回响应
    S-->>F: 返回内容
    F-->>U: 显示回复 + TTS 朗读

    Note over U,L: === 在线模式：拍照识花 ===
    U->>F: 拍摄花卉照片
    F->>F: Base64 编码图片
    F->>S: 调用 Edge Function (chat)
    S->>EF: 转发图片 + 请求
    EF->>AI: 调用 Qwen-VL-Plus 视觉模型
    AI-->>EF: 识别结果（花名/分布/花语）
    EF-->>S: 返回响应
    S-->>F: 返回识别结果
    F-->>U: 展示花卉信息卡片

    Note over U,L: === 离线模式：AI 不可用时 ===
    U->>F: 输入 "山茶花"
    F->>S: 调用 Edge Function
    S-->>F: ❌ 网络不可达 / 超时
    F->>L: 自动降级 → 关键词匹配
    L-->>F: 返回本地花库匹配结果
    F-->>U: 展示离线匹配结果
```
